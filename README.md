# 🏨 Manoj Grand Hotel Website

A simple and responsive hotel website built using **HTML** and **CSS**.
This project demonstrates basic web design concepts like layout, navigation, and styling.

---

## 📌 Features

* 🏠 Home Page with hotel introduction
* ℹ️ About Page with hotel details
* 🧭 Navigation bar for easy page switching
* 🎨 Styled using pure CSS (no frameworks)
* 🖼️ Background images and layout design
* 📱 Beginner-friendly structure

---

## 🛠️ Technologies Used

* HTML5
* CSS3

---

## 📂 Project Structure

```
📁 Manoj-Grand-Hotel
│── index.html
│── about.html
│── style (internal CSS used)
```

---

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the project folder
3. Double-click on `index.html`
4. Open in any browser (Chrome, Edge, etc.)

---

## ⚠️ Notes

* This project is created for **learning purposes**
* `<marquee>` tag is used but it is **deprecated** (not recommended in modern web development)
* Footer positioning may vary based on screen size
